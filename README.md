# Di-Paid – Bill Splitting Platform

Di-Paid is a full-stack web application designed to simplify bill splitting among users.

---

##  Tech Stack

### Frontend
- React
- HTML, CSS

### Backend
- Flask / FastAPI

### Database
- MySQL

---

##  Features

- Split bills among multiple users
- Clean and user-friendly interface
- Backend API integration

---

##  Project Structure

DiPaid/
├── di_paid_frontend/
├── di_paid_backend/
├── migrations/

---

##  Installation & Setup

### Clone the repository
git clone https://github.com/muhammad-habib513/Di-paid.git
cd Di-paid

---

### Backend Setup
cd di_paid_backend
pip install -r requirements.txt
uvicorn main:app --reload

---

### Frontend Setup
cd di_paid_frontend
npm install
npm run dev
